개요

어느덧 입사한 지 1개월이 되어간다. 몇년 근무한 것 같은 기분이 드는데 생각보다 시간이 많이 안 흘렀다.

지난 21년 연말 대표님께서 책 읽기 문화를 가지는 것이 어떻겠냐는 공약에 힘입어 개발 진행에 필요한 node.js와 관련된 기술 서적을 구매하게 되었다.

덤으로 독후감 작성도 요청하셨었음.

무작정 고른 책

솔직히 책을 구매하고 목차 부분을 제일 많이 읽은 것 같다. ㅋㅋㅋ.. 전반적으로 프레임워크에서 제공하는 기능과 대략적인 흐름 파악을 위해 구매한 도서이기에 어느정도 목적은 달성한 셈.

이렇게 목차만 보라고 사준 책이 아닐텐데..
레포트 제출하려고 책상에서 급하게 개요 펼치고 찍은 사진

이왕 구매한 것

전반적으로 기존 키핑 프로젝트 관련해서 소스코드를 분석중이기도 했고, 분석하며 습득한 기술을 실습해볼 프로젝트도 필요했다.
그래서 현재 키핑과 유사한 인프라를 구축하고 거기에 진행중인 프로젝트 소스코드를 올리기로 했다.

이왕 하기로 한 것. 도메인 구매하고 이것저것 AWS 서비스 신청하고~ 중간중간 인강도 들어가며 스터디에 몰두함. 사비도 좀 들었다.

여러 우여곡절이 많았음...

결과

백엔드, 프론트엔드, AWS(서버사이드 인프라)에서 뭔가 만들어서 배포할 수 있을 정도로 전반적인 스터디를 진행할 수 있었다.

지금 글을 작성하는 페이지도 소스코드 분석하며 습득한 기술로 만듦.

사이드 프로젝트 진행하며 사용한 기술
- 인프라 : Cloud-Formation, Route53, S3, Cloud-Front, Lambda, API-Gateway, EC2
- 백엔드 : node.js, express, nest.js
- 프론트엔드 : Angular
- 언어 : Typescript, Javascript

약속한 남은 일정까지 더 깊이있게 스터디 진행하고 간간히 결과를 공유할 수 있으면 좋겠다.
