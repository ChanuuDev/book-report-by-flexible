구매한 책 책 개요

node.js란

1. node.js 란
- node.js란 Chrome V8 Javascript 엔진으로 빌드된 Javascript 런타임임
※ Javascript 런타임 : 자바스크립트가 실행될 수 있는 환경을 말함. (ex/: 브라우저) JAVA로 치면 JVM 같은 환경

2. node.js의 활용처
- node.js는 node.js가 설치되는 환경이라면 개인PC, 서버, 웹 등 여러 환경에서 Javascript를 이용해 범용적인 소프트웨어 개발이 가능함
- javascript 생태계와 같이 발전하기 때문에 특히 웹 부분, 웹서버로서의 역할로 많이 발전해있음

3. 기반 프레임워크
- 백엔드에서 많이 사용되는 프레임워크는 Express, Nest.js 등이 있음

4. 키핑 서비스에서의 node.js
- 키핑 서비스에서의 node.js는 REST API로서 이용되고 있음
- 예를 들어 피드 등록, 피드 조회, 댓글 달기 등의 기능을 예로 들 수 있음
※ REST API란 간단하게 설명했을 때 HTTP 프로토콜을 이용하여 원격 자원의 CRUD(Create, Read, Update, Delete)를 가능하게 하는 API임
※ 조회, 수정, 삭제 등의 기능을 구현한 프로그램을 HTTP 통신을 이용해 사용함
